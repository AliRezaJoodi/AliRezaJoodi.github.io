### Features
- ATmega32 Microcontroller
- External Watchdog
- Voltage Reference with LM336
- 4-20mA or 1~5V Analog Input
- Zero Crossing Detector x3
- Triple Isolated Power Supply for SCR Drivers
- SCR Gate Driver (3x)
- Requires external Thyristor or Triac
- 1-Layer PCB

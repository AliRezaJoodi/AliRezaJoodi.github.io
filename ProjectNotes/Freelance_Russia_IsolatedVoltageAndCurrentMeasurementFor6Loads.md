### Features
- Isolated voltage measurement using HCPL-7840, 220V, 3 channels
- Isolated current measurement using CT, 40A, 6 channels
- ADC using MAX11198 (16-bit resolution, 2Msps, SPI Interface)
- 2-Layer PCB
